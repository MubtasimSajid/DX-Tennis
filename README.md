# DX-Tennis
DX Tennis, or DXT in short, in a simple 2-players single device game. The idea for it was to fusion the two popular games: DX Ball &amp; Ping Pong. It was created as our project submission for Structured Programming - 2 lab of CSE-'27 batch of IUT by the team of Mubtasim Sajid Ahmed, AKM Ishmam Tahmid, Mahmudul Hasan Sakib &amp; Obaidullah.
